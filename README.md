# Clojure Fractals

Implementations of various fractals for fun.

## Prerequisites

Use Clojure 1.4 and ensure you've leiningen installed.

## Installation

Clone the repository:
	
	git clone https://github.com/atc-/clojure-fractals.git

Then execute lein:
	
	lein run
